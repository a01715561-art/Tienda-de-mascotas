# Tienda-de-mascotas
El programa es una tienda de mascotas, la idea es que el usuario pueda registrar diferentes animales y ver la informacion de cada uno, para eso use una clase base llamada mascota, que tiene los datos generales, luego hice 3 clases que heredan de ella: perro, gato, pez, cada una con un atributo extra y un comportamiento propio.
El programa corre con un menu donde puedes elegir que tipo de mascota agregar, cada que se agrege una se crea un objeto dinamicamente y se guarda dentro de la clase tienda.
